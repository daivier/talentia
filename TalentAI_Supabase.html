TalentAI — Configuração do banco
Como funciona: O TalentAI usa Supabase como banco de dados PostgreSQL. Em poucos minutos você terá dados reais, autenticação e persistência completa.
1 Crie um projeto Supabase gratuito
Acesse supabase.com → New Project. Escolha um nome (ex: "talentai") e uma senha forte.
2 Execute o schema SQL
No Supabase, vá em SQL Editor → New Query e cole o conteúdo do arquivo supabase_schema.sql que acompanha este sistema. Clique em Run.
Isso cria 13 tabelas, views, RLS e dados de exemplo automaticamente.
3 Cole suas credenciais
Em Settings → API do seu projeto Supabase, copie a URL e a chave anon public.
Suas credenciais ficam salvas apenas no localStorage do navegador.
